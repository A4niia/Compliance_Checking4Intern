2.2 Formal Definitions
DEFINITION 1: Policy Semantics
Let P be a policy statement.
Let I be an interpretation (domain + assignment).
P is satisfied by I, written I ⊨ P, iff the conditions of P hold in I.

DEFINITION 2: FOL Translation Correctness
Let φ be the FOL translation of policy P.
φ is a correct translation iff:
  ∀I: I ⊨ P ↔ I ⊨ φ

DEFINITION 3: SHACL Implementation Correctness  
Let S be a SHACL shape implementing policy P.
Let G be a data graph.
S correctly implements P iff:
  G violates P ↔ G does not conform to S

DEFINITION 4: End-to-End Verification
The translation P → φ → S is verified iff:
  φ_src ≡ φ_shacl (logical equivalence)
  where φ_src is manual FOL and φ_shacl is SHACL2FOL(S)
2.3 Equivalence Checking Procedure
python# Pseudocode for equivalence checking

def verify_translation(fol_source, shacl_shape):
    """
    Verify that SHACL shape correctly implements FOL specification.
    
    Returns:
        - EQUIVALENT: Shapes are logically equivalent
        - STRONGER: SHACL catches more violations than intended
        - WEAKER: SHACL misses some violations
        - ERROR: With counterexample
    """
    
    # Step 1: Convert SHACL to FOL using SHACL2FOL
    fol_derived = shacl2fol(shacl_shape)
    
    # Step 2: Normalize both FOL formulas
    fol_src_normalized = normalize(fol_source)
    fol_der_normalized = normalize(fol_derived)
    
    # Step 3: Check equivalence using theorem prover
    # φ₁ ≡ φ₂  iff  (φ₁ → φ₂) ∧ (φ₂ → φ₁) is valid
    
    forward = implies(fol_src_normalized, fol_der_normalized)
    backward = implies(fol_der_normalized, fol_src_normalized)
    
    forward_valid = prove(forward)  # Using Vampire/Z3
    backward_valid = prove(backward)
    
    if forward_valid and backward_valid:
        return "EQUIVALENT"
    elif forward_valid and not backward_valid:
        counterexample = find_counterexample(backward)
        return ("STRONGER", counterexample)
    elif not forward_valid and backward_valid:
        counterexample = find_counterexample(forward)
        return ("WEAKER", counterexample)
    else:
        return ("INCOMPARABLE", find_counterexample(forward))
